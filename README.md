#  Hotel Reservation System

##  Description

Ce projet est une application web full-stack de gestion des hôtels et des réservations développée avec :

-  Spring Boot pour le backend
-  React JS pour le frontend
- MySQL pour la base de données

L’application permet :
- d’ajouter des hôtels
- de modifier et supprimer des hôtels
- d’effectuer des réservations
- d’afficher la liste des hôtels et réservations
- d’utiliser une recherche dynamique et une pagination

---

# Technologies utilisées

## Backend
- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL

## Frontend
- React JS
- Axios
- Bootstrap
